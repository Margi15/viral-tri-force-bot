# Viral Tri-Force Bot

Bot automatizado que:
1. Busca videos virales de historia en español en YouTube (50k+ vistas)
2. Analiza la fórmula viral con Claude AI
3. Crea scripts para YouTube Shorts + Facebook + Instagram
4. Genera videos MP4 con voz Edge TTS
5. Publica automáticamente 3x/día (9am, 5pm, 1am UTC)

## Secrets necesarios
- `YOUTUBE_API_KEY`
- `ANTHROPIC_API_KEY`
- `FACEBOOK_PAGE_TOKEN`
